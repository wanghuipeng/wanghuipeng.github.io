好用的图片懒加载插件 -- jquery.unveil.js

参考：

优蓝云代理官网   优蓝网wap站首页

说明：

从jquery.unveil.js的源码中可以看出，此js插件可以兼容jQuery.js和Zepto.js，也就是说，可以用在PC和移动端。


使用方法：

1.引入jquery.unveil.js
2.src里面加载占位图片，data-src里面加载真实的图片

3.添加引用方法
//图片在可视区域内加载
$(function(){

      $(".wrapper>.row>.col-md-12>img").unveil(1000);

});

这里的函数unveil内的值是每个img图片的高度。
